# Product Inventory Management (Frontend)

A clean and responsive **React (Class Components)** based user interface for managing product inventory.  
This frontend includes search, filtering, stock indicators, and a modern UI layout suitable for inventory dashboards.

---

## Features

### Search Products
Search by:
- Product **name**
- Product **category**

### Category Filtering
Dynamic category dropdown based on available product list.

### Stock Status Indicator
Color-coded labels:
- ** In Stock** — Stock > 0  
- ** Out of Stock** — Stock = 0  

###  Clean & Modern UI
- Rounded card layout  
- Spaced table rows  
- Soft shadows  
- Easy readability  
- Fully responsive design  

### CSV-Compatible Product Fields
Each product includes:
- `name`
- `unit`
- `category`
- `brand`
- `stock`
- `status` (In Stock / Out of Stock)
